# devnoder-cli

DevNoder CLI — Srvel Build Tools

## Install (after publishing)
```bash
npm install -g devnoder-cli
```

## Commands
```bash
devnoder init my-app --template react-vite-ts
devnoder build
devnoder deploy --project my-devnoder-app
devnoder status
devnoder doctor
```

## Dev
```bash
npm install
npm run build
node dist/index.js --help
```
